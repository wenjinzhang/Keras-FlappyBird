# Keras-FlappyBird

# Installation Dependencies:

* Python 3.6
* Keras
* pygame
* tensofflow
* scikit-image

# How to Run?

```
git clone https://github.com/wenjinzhang/Keras-FlappyBird.git
cd Keras-FlappyBird
python qlearn.py -m "Run"
```

If you want to train the network from beginning, delete the model.h5 and run qlearn.py -m "Train"
